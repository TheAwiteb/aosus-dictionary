# aosus-dictionary
معجم عربي للمساعدة في ترجمة البرمجيات إلى اللغة العربية.

# قبل الإسهام في التطوير 
نصب الأداة `jq` :

`sudo apt install jq`

شغل سكربت `prepare-devel.sh`

`sh prepare-devel.sh`
